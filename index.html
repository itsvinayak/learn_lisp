LISP Overview

  1. LISP was invented in 1958 by john McCarthy while he was at the Massachusetts Institute of Technology (MIT).
  2. it was first implemented by Steve Russell on IBM 704 computer
  3. it is particularly suitable for Artifical Intelligence Program, as it processes symbolic information effectively.
  4. Common Lisp originated, during the 1980s and 1990s, in an attempt to unify the work of several implementation groups that were successors to Maclisp, like ZetaLisp and NIL (New Implementation of Lisp) etc.
  5. Programs written in Common LISP do not depend on machine-specific characteristics, such as word length etc.
  6. CLISP is the GNU Common LISP multi-architechtural compiler used for setting up LISP
  7. LISP programs run either on an interpreter or as compiled code.
  8. LISP Uses Prefix Notation

                      (write (* 2 3))
