function

The defun construct is used for defining a function.
program to find area 
    (defun area(s)
       (format t "Area =  ~f" (* s s)))

    (area 4)
output 
    Area =  16.0
program to find max and min 
    (defun find_max_min(a b)
       (format t "max = ~a" (max a b))
       (format t "~%min = ~a" (min a b))
    )

    (find_max_min 22 4)
